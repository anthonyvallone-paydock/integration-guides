Direct Charge With 3ds

Credit Card (No Stored Details)​

Create a “Simple Widget” using the Paydock Client-SDK​

<form id="paymentForm">
    <div id="widget"></div>
    <input name="payment_source_token" id="payment_source_token" type="hidden">
</form>
<script src="https://widget.paydock.com/sdk/latest/widget.umd.min.js"></script>
<script>
    var widget = new paydock.HtmlWidget('#widget', 'publicKey');
    widget.onFinishInsert('input[name="payment_source_token"]', 'payment_source');
    widget.load();
</script>

Create a session vault token​

  1. Take the one time token from the payment_source_token field and create a session vault token
  2. Session vault token request
POST /v1/vault/payment_sources HTTP/1.1
Host: api.paydock.com
x-user-public-key: {{public-key}}
Content-Type: application/json

{
    "token": "6311c15f-8024-42f9-8ed5-200621ecfb1d",
    "vault_type": "session"
}

POST /v1/charges/3ds HTTP/1.1

Content-Type: application/json

Create a “3ds Canvas Widget”​

  1. Using the resource.data._3ds.token. Create a 3ds Canvas Widget
<form id="paymentForm">
    <div id="widget"></div>
    <input name="payment_source_token" id="payment_source_token" type="hidden">
</form>
<script src="https://widget.paydock.com/sdk/latest/widget.umd.min.js"></script>
<script>
    var widget = new paydock.Canvas3ds('#widget', 3ds-token);
    widget.on('chargeAuthSuccess', function (data) {
        console.log(data);
    });
    widget.on('chargeAuthReject', function (data) {
        console.log(data);
    });
    widget.on('chargeAuthCancelled', function (data) {
        console.log(data);
    });
    widget.on('additionalDataCollectSuccess', function (data) {
        console.log(data);
    });
    widget.on('additionalDataCollectReject', function (data) {
        console.log(data);
    });
    widget.on('chargeAuth', function (data) {
        console.log(data);
    });
    widget.load();
</script>
  1. Sample response from the 3ds canvas widget
{
    "event": "chargeAuth",
    "purpose": "secure3d",
    "message_source": "widget.paydock",
    "ref_id": "",
    "widget_id": "bbde2909-1606-4369-8ca2-f5a54c248033",
    "charge_3ds_id": "bbde2909-1606-4369-8ca2-f5a54c248033",
    "status": "authenticated"
}

Finalise the charge using the charge_3ds_id​

POST /v1/charges HTTP/1.1
Host: api.paydock.com
