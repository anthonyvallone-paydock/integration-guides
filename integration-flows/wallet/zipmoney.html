  • Using the one time token. Create a charge request

POST /v1/charges HTTP/1.1
Host: api.paydock.com
x-user-secret-key: {{secret-key}}
Content-Type: application/json

{
    "amount": "10.00", 
    "currency": "AUD",
    "token": "996280fe-961c-4f84-9e99-e490402c48b1"
}

when processing a payment from a checkout button, ensure the amount is the same as the checkout button init
