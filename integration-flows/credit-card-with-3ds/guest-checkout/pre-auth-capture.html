Pre Auth & Capture

Guest Checkout + 3ds​

Create a “Simple Widget” using the Paydock Client-SDK​


<form id="paymentForm">
    <div id="widget"></div>
    <input name="payment_source_token" id="payment_source_token" type="hidden">
</form>
<script src="https://widget.paydock.com/sdk/latest/widget.umd.min.js"></script>
<script>
    var widget = new paydock.HtmlWidget('#widget', public_key, 'not_configured');
    widget.onFinishInsert('input[name="payment_source_token"]', 'payment_source');
    widget.load();
</script>

Convert the OTT to a session vault token​

  • Take the OTT (One Time Token) from the payment_source_token field and create a session vault token

  • Session vault token request


POST /v1/vault/payment_sources HTTP/1.1
Host: api.paydock.com
x-user-secret-key: {{secret-key}}
Content-Type: application/json
{
    "token": "6311c15f-8024-42f9-8ed5-200621ecfb1d",
    "vault_type": "session"
}
  • Session vault response

{
    "status": 201,
    "error": null,
    "resource": {
        "type": "payment_source",

3ds Verification​

  • Using the session vault token or permanent vault token. Perform a 3ds pre auth request.

POST /v1/charges/3ds HTTP/1.1
Host: api.paydock.com
x-user-public-key: {{public-key}}
Content-Type: application/json

{
    "amount": "103",
    "currency": "AUD",
    "customer": {
        "payment_source": {
            "vault_token": vault_token,
            "gateway_id": "gateway-id"

},

  • Using the resource.data._3ds. Create a Canvas3ds widget using the client-sdk.

    <div id="widget"></div>
    <script src="https://widget.paydock.com/sdk/latest/widget.umd.min.js"></script>
    <script>
        var widget = new paydock.Canvas3ds('#widget', token);
        widget.setEnv('sandbox');
        widget.on('chargeAuthSuccess', function (data) {
            console.log(data);
        });
        widget.on('chargeAuthReject', function (data) {
            console.log(data);
        });
        widget.on('chargeAuthCancelled', function (data) {
            console.log(data);
        });
        widget.on('additionalDataCollectSuccess', function (data) {
            console.log(data);
        });
        widget.on('additionalDataCollectReject', function (data) {
            console.log(data);
        });
        widget.on('chargeAuth', function (data) {
            console.log(data);
        });
        widget.load();
    </script>

Create charge using the session vault token​

  • Using the vault_token. Create a charge request

Host: api.paydock.com
x-user-secret-key: {{secret-key}}
Content-Type: application/json

{
    "amount": "10.00",
    "currency": "AUD",
    "customer": {
        "payment_source": {
            "vault_token": "996280fe-961c-4f84-9e99-e490402c48b1"
        }
    }
}
