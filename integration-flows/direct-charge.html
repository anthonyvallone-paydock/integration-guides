Direct Charge

Credit Card (No Stored Details)​

Create a “Simple Widget” using the Paydock Client-SDK​

<form id="paymentForm">
    <div id="widget"></div>
    <input name="payment_source_token" id="payment_source_token" type="hidden">
</form>
<script src="https://widget.paydock.com/sdk/latest/widget.umd.min.js"></script>
<script>
    var widget = new paydock.HtmlWidget('#widget', 'publicKey');
    widget.onFinishInsert('input[name="payment_source_token"]', 'payment_source');
    widget.load();
</script>

Create a session vault token​

  1. Take the one time token from the payment_source_token field and create a session vault token
  2. Session vault token request
POST /v1/vault/payment_sources HTTP/1.1
Host: api.paydock.com
x-user-public-key: {{public-key}}
Content-Type: application/json

{
    "token": "6311c15f-8024-42f9-8ed5-200621ecfb1d",
    "vault_type": "session"
}
  1. Session vault response
{
    "status": 201,
    "error": null,
    "resource": {
        "type": "payment_source",
        "data": {
            "type": "card",
            "_source_ip_address": "x.x.x.x",
            "vault_type": "session",
            "expire_month": 12,

Create a permanent vault token​

  1. Take the one time token from the payment_source_token field and create a permanent vault token

  2. Permanent vault token request

POST /v1/vault/payment_sources HTTP/1.1
Host: api.paydock.com
x-user-public-key: {{public-key}}
Content-Type: application/json
{
    "token": "6311c15f-8024-42f9-8ed5-200621ecfb1d"
}

Create the Customer​

  1. Using the permanent vault token created earlier along with the customer details
POST /v1/customers HTTP/1.1
Host: api.paydock.com
x-user-public-key: {{public-key}}
Content-Type: application/json

{
    "first_name": "John",
    "last_name": "Smith",
    "email": "johnsmith@paydock.com",
    "phone": "+61413444444",
    "payment_source": {
        "vault_token": "3dc68807-d6d9-44bc-89a3-4cd922131b01"
    }
}
