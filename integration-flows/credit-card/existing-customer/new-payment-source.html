New Payment Source

Direct Charge​

Create a “Simple Widget” using the Paydock Client-SDK​


<form id="paymentForm">
    <div id="widget"></div>
    <input name="payment_source_token" id="payment_source_token" type="hidden">
</form>
<script src="https://widget.paydock.com/sdk/latest/widget.umd.min.js"></script>
<script>
    var widget = new paydock.HtmlWidget('#widget', public_key, gateway_id);
    widget.onFinishInsert('input[name="payment_source_token"]', 'payment_source');
    widget.load();
</script>

Convert the OTT to a permamnent vault token​

  • Take the OTT (One Time Token) from the payment_source_token field and create a permanent vault token

  • Permanent vault token request


POST /v1/vault/payment_sources HTTP/1.1
Host: api.paydock.com
x-user-secret-key: {{secret-key}}
Content-Type: application/json
{
    "token": "6311c15f-8024-42f9-8ed5-200621ecfb1d"
}
  • Permanent vault response

{
    "status": 201,
    "error": null,
    "resource": {
        "type": "payment_source",
        "data": {
            "type": "card",
            "_source_ip_address": "x.x.x.x",
            "vault_type": "permanent",
            "expire_month": 1,
            "expire_year": 2039,
            "card_name": "Wanda Mertz",
            "card_scheme": "mastercard",
            "card_number_last4": "0008",
            "card_number_bin": "51234500",
            "ref_token": "9139516499300184",
            "status": "active",
            "created_at": "2023-11-02T02:13:09.832Z",
            "company_id": "65235992df412413b29a8f3b",
            "vault_token": "3dc68807-d6d9-44bc-89a3-4cd922131b01",
            "updated_at": "2023-11-02T02:13:27.960Z"
        }
    }
}

Create customer with permanent vault token​

  • If there is no paydock customer created at this point. Please create a customer.
  • Create a customer using the permamnent vault token, gateway_id

POST /v1/customers HTTP/1.1
Host: api.paydock.com
x-user-secret-key: {{secret-key}}
Content-Type: application/json

{
    "first_name": "John",
    "last_name": "Smith",
    "reference": "external systems customer id",
    "email": "johnsmith@paydock.com",
    "payment_source": {
        "vault_token": "0fd20631-509e-45c7-a8f0-f1f36d6ad298",
        "gateway_id": gateway_id
    }
}

Add the new default payment​

  • Perform the following api call to add the default payment source to the customer

POST /v1/customers/:customer_id HTTP/1.1
Host: api.paydock.com
x-user-secret-key: {{secret-key}}
Content-Type: application/json

{
    "payment_source": {
        "vault_token": vault_token
    }
}

Perform a direct charge with the customer_id​


POST /v1/charges HTTP/1.1
Host: api.paydock.com
x-user-secret-key: {{secret-key}}
Content-Type: application/json

{
	"amount":"100",
	"currency":"AUD",
	"customer_id":"5e4bfbd86d244b438451fbd7"
}

Create a pre auth & capture using the chosen payment source​

  • Using the vault_token, gateway_id, _id (customer_id) that matches the payment_source_token. Create a charge request

POST /v1/charges?capture=false HTTP/1.1
Host: api.paydock.com
x-user-secret-key: {{secret-key}}
Content-Type: application/json

{
    "amount": "10.00",
    "currency": "AUD",
    "customer_id": customer_id,
    "customer": {
        "payment_source": {
            "vault_token": vault_token,
            "gateway_id": gateway_id
        }
    }
